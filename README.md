# Ecom

An Ecommerce app inspired from [OLX](http://olx.in/).

## Folder structure
| Folder name      | What is it ? |
| ----------- | ----------- |
| api | A GraphQL api built using nodeJs & typescript |
| app | Frontend built using Next Js |
| file-api | A file upload api using Rust |
