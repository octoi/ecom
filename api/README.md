# API 🕊️

Backend server code for **ECOM** .

This is made using `NODE JS`, `GRAPHQL`, & `PRISMA`.

## Setup
1 - Install dependencies
```bash
$ npm install
## Or yarn
$ pnpm install
```
2 - Prisma migrate
```bash
$ npm run migrate
## Or using yarn
$ pnpm migrate
```
3 - Run app
```bash
## Run with nodemon

$ npm run dev
## Or using yarn
$ pnpm dev

## Run without nodemon

$ npm run start
## Or using yarn
$ pnpm start
```
